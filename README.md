# Task 3 - Web Application Security Assessment

## Objective
The objective of this task was to set up and test a deliberately vulnerable web application (DVWA) for learning web application security concepts.

## Tools Used
- Kali Linux
- Apache2
- MariaDB
- PHP
- DVWA (Damn Vulnerable Web Application)

## Steps Performed
1. Installed and configured Apache2.
2. Installed and started MariaDB.
3. Installed PHP and required extensions.
4. Downloaded DVWA from GitHub.
5. Configured the DVWA database.
6. Created the database successfully.
7. Logged into DVWA.
8. Verified successful setup.

## Deliverables
- Setup Screenshots
- Screen Recording
- Project Report

## Learning Outcome
- Learned how to deploy a vulnerable web application.
- Understood Apache, MariaDB and PHP configuration.
- Successfully configured and accessed DVWA for security testing.
